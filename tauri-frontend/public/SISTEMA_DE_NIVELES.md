# ContextIA — Sistema de niveles y puntuación

## ¿Cómo funciona la puntuación?

Cuando respondés un ejercicio, la app envía tu respuesta a un modelo de inteligencia artificial que la evalúa del **0 al 100** teniendo en cuenta:

- **Gramática:** ¿la estructura de la oración es correcta?
- **Vocabulario:** ¿usaste la palabra o expresión adecuada?
- **Sentido:** ¿tu respuesta tiene el significado correcto en contexto?

Los sinónimos correctos también se consideran válidos. No es necesario acertar la respuesta exacta esperada.

## ¿Cómo subís de nivel?

Para subir al siguiente nivel necesitás **3 victorias consecutivas** (3 ejercicios aprobados seguidos sin fallo intermedio).

```
Aprobaste → Aprobaste → Aprobaste → ¡Subís de nivel!
Aprobaste → Aprobaste → Fallaste  → El contador de victorias vuelve a 0
```

---

## ¿Cómo bajás de nivel?

Si fallás **2 ejercicios consecutivos**, bajás al nivel anterior.

```
Fallaste → Fallaste → Bajás de nivel
Fallaste → Aprobaste → El contador de fallos vuelve a 0
```

Esto no aplica si ya estás en el nivel más bajo del idioma (por ejemplo A1 o HSK1).

## Niveles por idioma

Cada idioma tiene su propio sistema de niveles basado en los estándares internacionales:

### Inglés, Español, Portugués, Italiano, Ruso
Sistema **CEFR** (Marco Europeo Común de Referencia):

```
A1 → A2 → B1 → B2 → C1 → C2
(principiante)          (avanzado)
```

### Chino Mandarín
Sistema **HSK** (Hànyǔ Shuǐpíng Kǎoshì):

```
HSK1 → HSK2 → HSK3 → HSK4 → HSK5 → HSK6
(básico)                          (avanzado)
```

### Japonés
Sistema **JLPT** (Japanese Language Proficiency Test). Atención: en japonés los niveles van en orden **inverso** — N5 es el más básico y N1 es el más avanzado:

```
N5 → N4 → N3 → N2 → N1
(principiante)    (avanzado)
```

## ¿Qué muestra la sección "Métricas"?

- **Score en "Palabras difíciles":** es el último puntaje obtenido en un ejercicio donde esa palabra de vocabulario fue practicada. Cuanto más bajo, más difícil te resultó esa palabra.
- **Gráfico de actividad:** muestra los días en que hiciste ejercicios.
- **Nivel actual:** el nivel en el que estás practicando ese idioma ahora mismo.

## Consejos para progresar más rápido

1. **Guardá más vocabulario.** Los ejercicios se generan en base a tus palabras guardadas. Más variedad = ejercicios más ricos.
2. **No saltes niveles.** El sistema ajusta la dificultad automáticamente. Si bajás de nivel, es para consolidar la base.
3. **Reintentá los ejercicios fallidos.** Cada reintento no consume tu cuota de ejercicios gratuitos.
4. **Registrate para guardar tu progreso.** Sin cuenta, si cerrás el navegador perdés todo el historial de nivel y vocabulario.
